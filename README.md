# КОМПАС-3D AI Коннектор (Text-to-CAD)

Проект представляет собой пайплайн для автоматического параметрического твердотельного моделирования в САПР **КОМПАС-3D** с использованием Больших Языковых Моделей (LLM). Система транслирует промпты на естественном языке в структурированный JSON-формат, который затем интерпретируется Python-скриптом в нативные команды COM API КОМПАС-3D.

## 🚀 Особенности

*   **Текстовое моделирование**: Создание деталей путем описания их геометрии человеческим языком.
*   **Умная топология**: Автоматическое разрешение конфликтов геометрии (например, авто-определение плоских граней для эскизов, преобразование линий контура в осевые во избежание самопересечений при вращении).
*   **Гибкий графический интерфейс (GUI)**: Встроенная панель на базе Tkinter, позволяющая копировать системный промпт, вставлять сгенерированный JSON напрямую или загружать его из файла.
*   **Интеграция как макрос**: Приложение можно привязать к кнопке на ленте (Ribbon) в самом КОМПАС-3D.
*   **Модульная архитектура**: Легкое добавление новых CAD-операций за счет автоматической диспетчеризации команд через модуль `inspect`.


## 🔌 Запуск v0.1 (доступен в Releases) и Интеграция в интерфейс КОМПАС-3D (Создание кнопки) 

Чтобы вызывать AI Генератор прямо из окна КОМПАС-3D, вы можете добавить скомпилированный `.exe` файл в виде удобной кнопки на главную панель инструментов (Ribbon). Это штатный способ добавления сторонних утилит.

**Этап 1: Добавление утилиты в Конфигуратор**
1. Откройте КОМПАС-3D (рекомендуется создать любую Деталь, чтобы активировался основной интерфейс).
2. В верхнем текстовом меню выберите **Приложения** -> **Конфигуратор...**
   *(Если у вас скрыто текстовое меню, нажмите на значок «Шестеренка» (Настройки) в правом верхнем углу окна и найдите «Конфигуратор» там).*
3. В открывшемся окне Конфигуратора слева выберите раздел **Утилиты**.
4. Нажмите кнопку сверху окна **Состав - Добавить утилиты** (иконка с плюсиком или папкой).
5. В диалоге выбора файла (убедитесь, что в правом нижнем углу выбрано `*.exe` или `Все файлы`) найдите и выберите скомпилированный файл `gui_app\gui_app.exe`.
6. В открывшемся окне свойств:
   * **Имя:** напишите `AI Генератор` (или любое другое).
   * **Рабочая папка:** укажите папку с утилитой (обычно заполняется автоматически).
7. Нажмите **Сохранить конфигурацию**, и закройте Конфигуратор.

**Этап 2: Вывод кнопки на Ленту**
1. Нажмите **правой кнопкой мыши** в любом пустом месте на верхней ленте инструментов КОМПАСа.
2. В появившемся меню выберите **Настройка интерфейса...**
3. В верхней части открывшегося окна (в выпадающем списке)  найдите категорию **Приложения** или **Утилиты**.
4. Выберите эту категорию — в правой части окна появится ваш **AI Генератор**.
5. Зажмите значок `AI Генератора` левой кнопкой мыши и **перетащите его** прямо на любую рабочую панель ленты инструментов (например, рядом с кнопкой «Выдавливание»).
6. Нажмите зеленую галочку и закройте окно настроек интерфейса. 

🎉 **Готово!** Теперь вы можете вызывать интерфейс генератора в один клик прямо во время работы над деталью.


## 📂 Структура проекта

*   `main.py` / `gui_app.py` — Точки входа (CLI или GUI).
*   `kompas_connection.py` — Установка раннего связывания (Early Binding) и подключение к API 7-й версии КОМПАС-3D.
*   `json_runner.py` — Ядро интерпретатора. Читает JSON и мапит команды на соответствующие функции.
*   `operations.py` — Исполняемые CAD-операции (эскизы, выдавливание, вращение, фаски). Управляет `runner.ctx` (контекстом), запоминая активные эскизы и созданные объекты для ссылок (`use_feature`).
*   `prompt.txt` — Строгий системный промпт для LLM, задающий правила жизненного цикла эскизов и синтаксис JSON.

## 🛠 Требования

*   **САПР**: КОМПАС-3D v21.
*   **Python**: 3.8+ (по состоянию на 12.05.2026 версия python не выше 3.12)
*   **Библиотеки**: `pywin32` (необходима для работы с COM-объектами)

## 📦 Установка и запуск исходного кода

1. **Клонируйте репозиторий и перейдите в папку проекта:**
   ```bash
   git clone <URL_репозитория>
   cd ProjectKOMPAS3D
   ```

2. **Создайте виртуальное окружение и установите зависимости:**
   ```bash
   python -m venv venv
   # Активация для Windows (PowerShell):
   .\venv\Scripts\Activate.ps1
   # Установка pywin32
   pip install pywin32
   ```

3. **Запуск графического интерфейса:**
   ```bash
   python gui_app.py
   ```


## 📝 Как составить запрос

1. В графическом интерфейсе нажмите **«Скопировать промпт»**.
2. Вставьте его как *System Prompt* (или первое сообщение) в любую мощную LLM (рекомендуется GPT-4o, Claude 3.5 Sonnet или подобные модели с сильной логикой).
3. Напишите свой запрос, например:
   > *"Создай вал длиной 100 и диаметром 20. На торце сделай вырез квадратной формы 10х10 глубиной 15."*
4. Скопируйте ответный JSON, вставьте его в текстовое поле GUI-приложения и нажмите **«Построить из текста»**.

## 🏗 Поддерживаемые CAD-операции (JSON)

Текущая версия ядра поддерживает следующие операции (массив `operations`):
*   `create_document_3d`, `save_document`
*   `create_sketch_on_plane`, `create_sketch_on_face`
*   `draw_line`, `draw_rectangle_center`, `draw_circle`, `draw_polygon`
*   `finish_sketch`, `close_contour`
*   `extrude_sketch` (выдавливание), `extrude_cut` (вырез), `revolve_sketch` (вращение)
*   `chamfer_all_edges` (фаска), `fillet_all_edges` (скругление), `add_thread` (резьба)

*Примечание: для замыкающих линий вращения (лежащих на осях) должен передаваться параметр `"is_axis": true`.*
pyinstaller --noconfirm --onedir --windowed --paths "." --add-data "prompt.txt;." gui_app.py
